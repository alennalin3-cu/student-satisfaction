# US News Top Public US Universities Data (Publicly Available)

## Overview
This repository contains a compiled dataset of publicly available university data, collected and cleaned from U.S. News and other publicly accessible sources. The goal of this dataset is to enable independent analysis of factors related to university rankings and institutional characteristics.

Only non-proprietary, publicly available information is included.

## Data Source
All data in this repository was collected from publicly accessible sources, including:
- U.S. News public university profiles
- University websites
- Common data sets

No paid, scraped, or restricted data is included.

## Files
- `US_NEWS_Public_Data.csv`: Main dataset containing university-level information.

## Dataset Description
The first row lists all included categories in the file.
Each row represents a single university. Columns include metrics related to institutional characteristics, rankings, and resources.
Included data is both quantitative and qualitative, with both sources being collected as recently as August 2025. 

### Example Columns
- `University_Name`
- `Public_Rank`
- `In-State Tuition`
- `Acceptance_Rate`
- `Percent_Awarded_Pell`
- `Reading_Days`

## Usage
This dataset is intended for:
- Academic projects
- Portfolio analysis
- Exploratory data analysis
- Educational and research purposes

If you use this data, please cite the original public sources.

## Limitations
- Rankings and metrics may change year to year.
- Data accuracy depends on the original public sources.
- This dataset does not include proprietary or paywalled information.
